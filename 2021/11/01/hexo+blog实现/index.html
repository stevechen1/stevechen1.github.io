hexo+github+blog的实现

1. hexo官网

官网

可以直接根据官网教程

2. 安装 hexo-cli

npm install hexo-cli -g

需要node.js支持 v16.14.0

3. 配置github

github创建仓库 stevechen1.github.io

4.主题配置

我用的主题是fluid
fluid

安装

npm install --save hexo-theme-fluid

配置
在_config.yml文件中改
theme: fluid

5.hexo指令

hexo init 初始化
hexo new 'title'
hexo clean
hexo g
hexo d
hexo s

6.个人域名

企鹅云上购买域名

配置域名 进入dnspod

在source文件家中新建CNAME文件

用vscode 打开 写入域名 例如我就写入12am.top 保存

执行hexo d 提交到github

打开12am.top 完成
