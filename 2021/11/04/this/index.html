this的定义

this的指向是不确定的，this就是属性或方法“当前”所在的对象（运行环境，找到此时this所在的环境，this代表的就是上层对象）
多层this

var o = {
f1: function () {
console.log(this);
var f2 = function () {
console.log(this);
}();
}
}
o.f1()
// Object
// Window
第一个指代的是o ,第二个指代的是window(非严格模式)

解决办法

绑定_this

var _this = this
将对象绑定在_this里面
